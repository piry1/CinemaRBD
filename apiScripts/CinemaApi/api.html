CINEMA API

GET

link opis
cinemaapi.php/bilety/[id] Zwraca wszystkie bilety bądź jeden wybrany.
cinemaapi.php/filmy/[id] Zwraca wszystkie filmy bądź jeden wybrany.
cinemaapi.php/miejsca/[id] Zwraca wszystkie miejsca bądź jedno wybrane.
cinemaapi.php/rezerwacje/[id] Zwraca wszystkie rezerwacje bądź jedną wybraną.
cinemaapi.php/sale/[id] Zwraca wszystkie sale bądź jedną wybraną.
cinemaapi.php/seanse/[id] Zwraca wszystkie seanse bądź jeden wybrany.
cinemaapi.php/bilet/id Zwraca informacje o bilecie
cinemaapi.php/seansefilmu/idFilmu Zwraca wszystkie seanse dla danego filmu
cinemaapi.php/wolnemiejsca/idSeansu Zwraca wszystkie wolne miejsca dla danego seansu
cinemaapi.php/biletyusera/id Zwraca wszystkie bilety danego użytkownika
